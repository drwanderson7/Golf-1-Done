How to Use

Overview

Tracks picks across 4 one-and-done pools for the 2026 PGA Tour season — Sony Open (Jan 15) through BMW Championship (Aug 20). Each player can only be used once per pool across the season.

Importing Rankings

  • Go to datagolf.com/datagolf-rankings
  • Click the Export / CSV button on the page
  • Click ↑ Import Rankings in the header (or drop the file in the box above)
  • Rankings update instantly — names, DG rank, OWGR, tour, and rating are all pulled from the file
  • Rankings persist across browser sessions until you import a newer file

My Picks tab

Dropdowns list players by DG rank. Already-used players show ✗ in red. Duplicate picks in the same pool get a conflict warning.

Pick Distribution tab

Shows how your picks spread (or stack) across pools each week. Bar color: blue = 1 pool, yellow = 2, orange = 3, red = all 4.

Remaining Players tab

All players still available in each pool ordered by DG rank. Filter to a single pool to plan ahead.

DG Rankings tab

Full imported rankings with search, rank filters, and a PGA Tour–only filter. Dimmed rows = player already used in at least one pool.

Pool names

Edit pool names in the text fields at the top of the My Picks tab. Saves automatically.

Export & Import picks

  • Click ↓ Export Picks CSV to save all picks across all 4 pools as a backup file. Do this regularly.
  • Click ↑ Import Picks to restore picks from any previously exported CSV. This will overlay the imported picks on top of whatever is currently saved — existing picks not in the file are kept.
  • Pool names are also restored from the CSV header row automatically.

Data storage

Picks, pool names, and the imported player list are all saved in browser localStorage — they persist across browser sessions on the same machine. If you ever clear browser data or move computers, use ↓ Export Picks CSV first so you can restore with ↑ Import Picks.
